# WanderAI

**WanderAI** is an AI-powered travel itinerary generator built with Next.js. It helps users plan their perfect trips by leveraging artificial intelligence to create personalized schedules based on their preferences, budget, and destination.

---

## 🚀 Key Features

* **AI-Generated Itineraries:** Custom travel plans tailored to user input.
* **Google Authentication:** Secure and seamless login using NextAuth.js.
* **Modern Dashboard:** A clean interface for managing and viewing saved trips.
* **Fully Responsive:** Optimized for a flawless experience on Mobile, Tablet, and Desktop.
* **Dark/Light Mode:** Native support for theme switching.

---

## 🛠 Tech Stack

* **Frontend:** Next.js 15 (App Router), React, TypeScript
* **Styling:** Tailwind CSS, Shadcn UI
* **Authentication:** NextAuth.js
* **Backend:** Node.js / Express (API Integration)
